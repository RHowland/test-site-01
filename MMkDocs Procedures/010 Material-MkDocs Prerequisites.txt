WorkStep 035 
Install Python
----------------------------------------------
websvr01 -   UserName: 
+++++++++++++++++++++++++++++++++++++

    a. Install Chocolately 

>   Set-ExecutionPolicy Bypass -Scope Process                       
    ----------------------------------

    Expected Response: none
	
>   Set-ExecutionPolicy Bypass -Scope Process -Force; 
    [System.Net.ServicePointManager]::
	SecurityProtocol = [System.Net.ServicePointManager]::
	SecurityProtocol -bor 3072; 
	 iex ((New-Object System.Net.WebClient).DownloadString
    ('https://community.chocolatey.org/install.ps1'))	
	
	Please note, the command has to be pasted in a single line.
	To do so paste and copy it via a text editor before pasting it into
	powershell.
	
	b. Confirm Chocolately installation was successful

>   choco --version                       
    ----------------------------------
	
	Expected Response: 2.5.0
	
	This version may differ slightly on your system.
	
	c. Install Python 

>   choco install python -y                        
    ----------------------------------

    Expected Response: none
	
	d. Confirm Python installation was successful
    
>   python --version                       
    ----------------------------------
	
	Expected Response: Python 3.13.7

	This version may differ slightly on your system.

    e. Install vscode:

>   choco install vscode -y          
    -----------------------------------------------

    Expected Response: Installing the following packages:
    vscode
    By installing, you accept licenses for the packages.
    Downloading package from source 
	'https://community.chocolatey.org/api/v2/'
    Progress: Downloading vscode.install 1.105.1... 100%

    f. Confirm vscode installation was successful 

>   code --version                       
    ----------------------------------
	
	note that 'v' is left out when calling vscode from powershell 
	after chocolatey installation
	
	Expected Response: 1.105.1
    ....x64

    g. Install git:

>   choco install git -y
    --------------------------------------

    Expected Response: 
	
	h. Confirm git installation was successful

>   git --version 
    --------------------------------------

    Expected Response: git version 2.48.1.windows.1
	
	i. Install notepad++

>   choco install notepadplusplus -y
    --------------------------------------

    Expected Response: 
	
	h. Confirm git installation was successful

>   notepad++ 
    --------------------------------------

    Expected Response: the notepadplusplus gui will open
